# SpaExperiments

    几款单页方案的简单demo

## 目录结构

```
├── README.md
├── bower.json   // bower 依赖管理
├── .bowerrc  // bower配置，主要配置依赖包保存目录
├── deploy       // 打包配置文件(打包未做)
├── pub  // 打包输出路径(打包未做)
├── src  // 源码目录
│   ├── css  // css文件目录
│   ├── javascript  // js文件目录
│   ├── view  // 页面入口目录
├── res  // 静态资源目录


```

## 运行说明

- 项目根目录下，bower install

- 如全局安装了puer，根目录下运行puer

- 页面入口/src/view/regular-state-demo/index.html
